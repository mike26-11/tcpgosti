# Progetto Client-Server TCP

## Cosa fa il progetto

Questo progetto mostra come funziona una comunicazione tra un **client** e un **server** usando TCP.

* Il server aspetta una connessione.
* Il client si collega al server.
* Il client può mandare un messaggio.

È un esempio semplice per capire come funziona la rete in Java.

---

## Classi del progetto

### Server

La classe `Server`:

* Apre una porta (3000)
* Aspetta un client
* Legge un messaggio
* Può chiudersi

---

### Client

La classe `Client`:

* Si collega al server
* Invia un messaggio ("CIAO")

---

### MainServer

La classe `MainServer` serve per avviare il server.

Contiene il metodo `main`.

---

### MainClient

La classe `MainClient` serve per avviare il client.

Contiene il metodo `main`.

---

## Requisiti

* Java installato
* Un programma per eseguire il codice (IDE o terminale)

---

## Scopo

Questo progetto serve per imparare:

* Cos'è un server
* Cos'è un client
* Come funziona una connessione TCP
